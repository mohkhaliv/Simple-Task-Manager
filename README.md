# Advanced Basic Task Manager

Spring Boot + Angular + MySQL learning project.

## Backend
1. Create database `taskdb`
2. Update `backend/src/main/resources/application.properties` if needed
3. Run:
   - Windows: `mvnw.cmd spring-boot:run`
   - Mac/Linux: `./mvnw spring-boot:run`

## Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

Frontend: `http://localhost:4200`
Backend: `http://localhost:8080/api/tasks`
